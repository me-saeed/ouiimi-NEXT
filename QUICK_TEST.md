# Quick Test Guide - 5 Minute Test

## As Business Owner (2 minutes)

1. **Sign Up & Register Business**
   - Go to `/signup` → Create account
   - Go to `/business/register` → Register business

2. **Add Service with Time Slots**
   - Dashboard → List tab → Add Service
   - Fill form → Create
   - Edit service → Add Time Slot → Save

3. **View Dashboard**
   - Check Bookings, Services, Staff tabs work

## As Customer (3 minutes)

1. **Sign Up**
   - Go to `/signup` → Create different account

2. **Book a Service**
   - Home page → Click service → Select date/time → Book Now
   - Cart → Review → Book Now
   - Confirmation page appears ✅

3. **Manage Booking**
   - Go to `/profile` → View booking → Test Reschedule/Cancel

**Done!** If all above work, the core flow is functional.
